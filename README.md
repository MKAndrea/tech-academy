# Tech-Academy

## Descrizione del progetto
Tech-Academy è una piattaforma didattica moderna progettata per presentare un'offerta formativa d'eccellenza nel settore informatico. Il sito funge da vetrina per corsi specializzati in sviluppo web, programmazione e cybersecurity.

## Sezioni del sito
* **Header**: Contiene il branding aziendale e la navigazione principale.
* **Chi Siamo**: Una breve introduzione alla missione dell'accademia.
* **I nostri corsi**: Visualizzazione a griglia dei percorsi formativi disponibili.
* **Perché sceglierci**: Elenco puntato dei vantaggi competitivi della nostra academy.
* **Footer**: Informazioni di copyright e chiusura pagina.

## Principali comandi Git utilizzati
Durante lo sviluppo sono stati utilizzati i seguenti comandi per la gestione del versionamento:

| Comando | Descrizione |
| :--- | :--- |
| `git branch` | Elenca, crea o elimina i branch del repository. |
| `git checkout -b` | Crea un nuovo branch e vi si sposta immediatamente. |
| `git add .` | Aggiunge le modifiche dei file all'area di stage. |
| `git commit -m` | Registra i cambiamenti localmente con un messaggio descrittivo. |
| `git merge` | Unisce la cronologia di due branch differenti. |
| `git push` | Invia i commit locali al repository remoto su GitHub. |
